# PNA 서비스 로드맵

> PNA(Penguins Never Alone) — 대학 동문 기반 프로덕트 직군 네트워크 및 테크 콘텐츠 플랫폼
> 이 문서는 **서비스 비전과 버전별 목표**를 정리합니다. 실제 구현 현황은 `개발_현황.md` 참고.

---

## 비전

**"대학 동문이라는 신뢰를 기반으로, 프로덕트 직군이 서로를 발견하고 함께 성장하는 네트워크"**

프로덕트 매니저·디자이너·데이터 분석가 등 프로덕트 직군은 현업에서도 링크드인에서도 같은 대학 동문을 만나기 어렵다는 문제에서 출발. AI·실리콘밸리 트렌드를 중심으로 콘텐츠를 아카이빙하고, 동문 간 토론·네트워킹·추천 채용을 지원한다.

---

## 서비스 구성 (북극성)

1. **콘텐츠 아카이브 + 토론 (Knowledge)** — 복수 자료 연결, 카테고리·태그, 토론
2. **온톨로지** — 콘텐츠·개념·인물 간 지식 그래프
3. **AI Now** — AI 트렌드·논문 요약·타임라인
4. **실리콘밸리 트렌드** — 최신 소식 큐레이션
5. **디지털 명함 & 온보딩** — 펭귄 캐릭터 + NFC 명함
6. **커뮤니티 확장** — 꼬리미션 챗봇, 오픈채팅, 사푼사프 연계
7. **오프라인 워크샵** — 프레임워크 체험형 AI 워크샵

## 버전 로드맵

### ✅ v0.1 — 플랫폼 초석
- Next.js/Supabase 초기 셋업, 아이템 등록·목록 구현
- Auth (Supabase + Google OAuth)
- 기본 레이아웃(GNB, About, Open Chat)

### 🚧 v0.2 — Knowledge MVP + 오프라인 워크샵 (현재)
- 아이템 상세·댓글·OG 메타데이터
- 라이트/다크 디자인 시스템
- 오프라인 워크샵 챗봇 (`26-1q-workshop`): 4개 도메인, 6단계 Phase, VIP 만족도, 스트리밍, 세션 저장
- 프레임워크 유형 테스트 페이지
- Knowledge RAG (워크샵 챗봇에 한정)
- 오픈채팅 랜딩 (정적 3개 대학)

👉 세부 구현 상태는 `개발_현황.md` 참고.

### 🔧 v0.2.1 — Google OAuth / 세션·관리자 권한 정비 (v0.3 선행 필수)
**목표**: v0.2의 인증 결함을 해결하고 관리자 권한 체계를 도입한다.

- `/auth/callback` 라우트 + PKCE 코드 교환
- `middleware.ts`로 세션 자동 갱신
- 서버 Supabase 클라이언트 쿠키 `getAll/setAll` 핸들러
- `profiles` 테이블 + `is_admin` 플래그, Google 이름·아바타 동기화
- 로그인 후 원래 페이지로 복귀(`next` 파라미터)

기획·테스트 결과 → `docs/v0.2.1_google_oauth_debugging.md`

### 🎯 v0.3 — Knowledge Graph (Categories, Tags & Ontology)
**목표**: Knowledge를 단순 목록에서 "카테고리·태그로 필터 가능하고 개념·인물이 엮인 지식 그래프"로 전환한다.

- Part A — 카테고리·태그 체계 (위계형 카테고리 + 평면 태그 + 필터·랜딩)
- Part B — 온톨로지 (통일 노드·엣지, `/ontology` 탐색, 관리자 편집 UI, 아이템 상세 "연결된 개념")
- 기획 문서 → `docs/v0.3_knowledge_graph.md`

### 📋 v0.4 이후 — 확장 영역 (범위 미확정)
아래 항목들은 v0.4 이후 우선순위에 따라 배치된다.

- **Knowledge 심화**
  - 복수 자료 연결 (한 아이템에 여러 URL 묶기, `parent_id` 활용)
  - 오픈채팅 대화 → 아이템 아카이빙 파이프라인
  - Knowledge 전역 RAG 검색
- **AI Now / 실리콘밸리 트렌드**
  - 논문·뉴스 자동 수집·요약 파이프라인
  - 타임라인 뷰
  - 토픽별 구독·알림
- **디지털 명함 & 멤버 네트워크**
  - 펭귄 캐릭터 커스터마이저
  - 회원 프로필·경력 카드
  - 동문 간 레퍼럴 채용
  - NFC 명함·포토카드 실물 연동
- **커뮤니티 확장**
  - 꼬리미션 챗봇(신규 온보딩)
  - 사푼사프 등 외부 커뮤니티 연계
  - 정기 오프라인 워크샵 고도화

## 기술 방향성

- **디자인 시스템**: Tailwind v4 + CSS 변수 기반 라이트/다크 토큰
- **분석**: GTM / GA4, SEO·GEO 최적화
