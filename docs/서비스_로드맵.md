# PNA 서비스 로드맵

> PNA(Penguins Never Alone) — 대학 동문 기반 프로덕트 직군 네트워크 및 테크 콘텐츠 플랫폼
> 이 문서는 **서비스 비전과 버전별 목표**를 정리합니다. 실제 구현 현황은 `개발_현황.md` 참고.

---

## 비전

**"대학 동문이라는 신뢰를 기반으로, 프로덕트 직군이 서로를 발견하고 함께 성장하는 네트워크"**

프로덕트 매니저·디자이너·데이터 분석가 등 프로덕트 직군은 현업에서도 링크드인에서도 같은 대학 동문을 만나기 어렵다는 문제에서 출발. AI·실리콘밸리 트렌드를 중심으로 콘텐츠를 아카이빙하고, 동문 간 토론·네트워킹·추천 채용을 지원한다.

---

## 서비스 구성 (북극성)

1. **Knowledge** — 콘텐츠 아카이브 + 지식 그래프 (List / Graph 뷰 전환)
2. **Events** — 오프라인 프로그램 허브 (컨퍼런스, 워크샵, 정기모임, 스터디)
3. **Career** — 추천 채용 네트워크 + 이직 스터디 + 커리어 성장
4. **디지털 명함** — 회원 프로필 카드 + NFC 명함 연동
5. **AX Lab** — 프레임워크 체험형 AI 워크샵
6. **커뮤니티** — 대학별 오픈채팅, 사푼사프 연계

## 버전 로드맵

### ✅ v0.1 — 플랫폼 초석
- Next.js/Supabase 초기 셋업, 아이템 등록·목록
- Auth (Supabase + Google OAuth)
- 기본 레이아웃(GNB, About, Open Chat)

### ✅ v0.2 — Knowledge MVP + 오프라인 워크샵
- 아이템 상세·댓글·OG 메타데이터
- 라이트/다크 디자인 시스템
- AX Lab 워크샵 챗봇 (4개 도메인, 6단계 Phase, Gemini 스트리밍)
- Knowledge RAG, 프레임워크 유형 테스트

### ✅ v0.2.1 — Google OAuth / 세션·관리자 권한 정비
- `/auth/callback` + PKCE, `proxy.ts` 세션 갱신
- `profiles` 테이블 + `is_admin`, 쿠키 `getAll/setAll`

### ✅ v0.3 — Events
- `/events` 메인 (히어로 캐러셀, 예정/진행 중/지난 행사)
- `/events/[slug]` 상세 (신청 CTA, 갤러리, 후기)
- 노션 '오프라인 모임' DB 기반 행사 8건 등록

### ✅ v0.4 — Knowledge Graph
- Part A: 카테고리·태그 체계 (위계형 카테고리, 평면 태그, 필터, 자동완성)
- Part B: 온톨로지 (react-flow 그래프, concept/person 시드, 아이템 상세 "연결된 개념")
- Knowledge 홈에 List / Graph 뷰 토글 통합

### 🎯 v1.0 — 정식 출시
**목표**: 현재까지 구축한 기능을 안정화하고 프로덕션 품질로 다듬어 정식 출시한다.

- 커스텀 도메인 연결 + SSL
- SEO 최적화 (sitemap, robots.txt, 구조화 데이터)
- 성능 최적화 (이미지 최적화, 캐싱 전략, Core Web Vitals)
- 에러 핸들링·빈 상태 UI 정비
- GTM / GA4 분석 연동
- 프로덕션 모니터링 (Vercel Analytics)
- README·About 페이지 정식 콘텐츠
- Supabase 보안 감사 (RLS 정책, API Key 노출 점검)
- 기획 문서 → `docs/v1.0_launch.md` (TBD)

### 📋 v1.1 — Career
**목표**: 동문 네트워크 기반의 커리어 성장 플랫폼을 구축한다.

- `/career` 메인 (추천 채용 보드 + 이직 스터디 + 커피챗)
- 추천 채용 네트워크 (동문 레퍼럴)
- 이직 스터디 관리 (모집·커리큘럼·과제·환급 트래킹)
- 커피챗 매칭 (직무·목표 기반)
- 기획 문서 → `docs/v1.1_career.md`

### 🃏 v1.2 — 디지털 명함
**목표**: PNA 멤버의 정체성을 카드 한 장으로 표현하고, 오프라인과 연결한다.

- `/profile` 편집 + 공개 프로필 페이지
- 펭귄 캐릭터 커스터마이저
- 디지털 명함 카드 (이름·직무·회사·경력·관심사)
- NFC 명함/포토카드 실물 연동
- 기획 문서 → `docs/v1.2_digital_card.md`

### 📋 v2.0+ — 확장 영역 (범위 미확정)
- Knowledge 심화 (복수 자료 연결, RAG 전역 검색)

## 성공 지표 (North Star 후보)

- **주간 활성 토론**: 주간 신규 댓글이 달린 아이템 수
- **동문 간 연결**: 월간 오픈채팅 신규 가입 / 온오프라인 이벤트 참가
- **콘텐츠 깊이**: 아이템당 연결된 자료 수 (복수 자료 구조 차별화 지표)

## 기술 방향성

- **Next.js 16 + React 19** (Turbopack)
- **Supabase**: Auth / PostgreSQL / RLS / Storage
- **AI**: Google Generative AI (Gemini 2.5 Flash) 중심, 필요 시 멀티모델
- **그래프**: @xyflow/react (온톨로지)
- **디자인 시스템**: Tailwind v4 + CSS 변수 기반 라이트/다크 토큰
- **분석**: GTM / GA4, SEO·GEO 최적화
