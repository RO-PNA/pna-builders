## 분석 기준

### 핵심 지표

| 지표 | 계산 방법 | 목표 |
|------|----------|------|
| 전체 만족도 | Q1 평균 | 4.0 이상 |
| 게임 재미도 | Q6 평균 | 4.0 이상 |
| 학습 효과 | Q12 평균 | 3.5 이상 |
| 실무 적용 가능성 | Q13 평균 | 3.5 이상 |
| NPS | (추천자% - 비추천자%) | 30 이상 |
| 재참여 의향 | Q18 평균 | 4.0 이상 |

### NPS 계산
- 추천자 (Promoter): 9~10점
- 중립 (Passive): 7~8점
- 비추천자 (Detractor): 0~6점
- NPS = 추천자% - 비추천자%

### 크로스 분석
- 프레임워크 유형별 만족도 차이
- 도메인별 몰입도 차이
- 난이도 인식 vs 학습 효과 상관관계
