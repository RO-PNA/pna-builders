# PNA 개발 현황

> 현재 개발된 기능·페이지·인프라를 **실제 구현 상태 그대로** 추적한다.
> 새 기능이 머지되면 해당 섹션을 갱신한다. 계획·미구현 항목은 `서비스_로드맵.md`로 분리.
> **현재 버전**: v0.4 (개발 중) / **마지막 업데이트**: 2026-04-18

---

## 1. 배포·인프라

| 항목 | 상태 |
|------|------|
| 호스팅 | Vercel (`pna-builders.vercel.app`) |
| 프레임워크 | Next.js 16 + Turbopack |
| React | 19 |
| 스타일 | Tailwind CSS v4 + `@tailwindcss/typography` |
| DB / 인증 | Supabase (PostgreSQL + Auth + RLS) |
| AI 모델 | Google Generative AI — `gemini-2.5-flash` (스트리밍) |
| 마크다운 렌더링 | `streamdown` (AI 응답용) |
| 그래프 시각화 | `@xyflow/react` (온톨로지) |
| 기타 라이브러리 | `react-nice-avatar`, `recharts`, `@supabase/ssr` |

### Supabase 스키마 (확인된 테이블)
- `items` — 콘텐츠 아카이브 (title, url, summary, author, category_id, points, parent_id, created_by)
- `comments` — 아이템 댓글 (item_id, user_id, author_name, content)
- `categories` — 위계형 카테고리 (slug, name, parent_id, sort_order) ✅ v0.4
- `tags` — 평면형 태그 (slug, name, usage_count) ✅ v0.4
- `item_tags` — 아이템↔태그 다대다 ✅ v0.4
- `ontology_nodes` — 지식 그래프 노드 (node_type, ref_id, slug, name, description) ✅ v0.4
- `ontology_edges` — 지식 그래프 엣지 (from_node, to_node, relation, weight) ✅ v0.4
- `profiles` — 사용자 프로필 (display_name, avatar_url, is_admin) ✅ v0.2.1
- `workshop_sessions` — 워크샵 팀 세션
- `workshop_messages` — 워크샵 대화 로그

## 2. 인증 — 구글 로그인 (v0.2.1 완료)

- Provider: **Google OAuth 단일** (Supabase PKCE 플로우)
- `/auth/callback` 라우트 — `exchangeCodeForSession` + `next` 파라미터 복귀
- `src/proxy.ts` — Next.js 16 proxy 컨벤션으로 세션 자동 갱신
- 서버 쿠키 `getAll/setAll` 핸들러
- `profiles` 테이블 + `is_admin` 플래그 + Google 메타 동기화 트리거
- Supabase Redirect URL 설정 완료

## 3. 페이지별 구현 현황

### GNB 메뉴 구조
`Knowledge | Events | Career | AX Lab | Ontology | About` + Login/Logout (모두 앞글자 대문자)

### `/` — Knowledge 홈
- **구현 완료** (v0.4 업데이트)
- 카테고리 칩 필터 (전체 / AI / Product / Engineering / Business / Community)
- 인기 태그 표시 + 태그 필터링
- 쿼리스트링 기반 필터 (`?category=ai&tag=rag`)
- Supabase 쿼리 4단계 병렬화 (Promise.all)
- `loading.tsx` — 필터 전환 시 스피너 표시
- 사이드바: 최신 댓글 10개

### `/item/[id]` — 아이템 상세
- **구현 완료** (v0.4 업데이트)
- OG 메타데이터, 댓글, 마크다운 복사
- **연결된 개념 섹션** — 온톨로지 노드 1-hop 이웃 카드 표시 ✅ v0.4

### `/submit` — 아이템 등록
- **구현 완료** (v0.4 업데이트)
- 2단계 카테고리 드롭다운 (대분류 → 소분류)
- 태그 자동완성 입력 (최대 5개, Enter 추가, 기존 태그 서제스트)

### `/events` — Events 허브
- **구현 완료** (v0.3)
- 히어로 캐러셀 (4초 간격, 3개 자동 순환, 호버 일시정지)
  - 우선순위: 예정 → 진행 중 → 최근 종료
- 예정 행사 / 진행 중인 행사 / 지난 행사 3단 구조
- 카테고리 배지 (Conference / Workshop / Meetup / Study / Social)
- 상태 배지 (모집 중 / 마감 임박 / 진행 중 / 종료됨)
- 등록된 행사 8건 (노션 '오프라인 모임' DB 연동)

### `/events/[slug]` — 행사 상세
- **구현 완료** (v0.3)
- 행사 정보 바 (날짜/시간/장소/가격)
- 신청 CTA (외부 링크) / 종료됨 안내
- 마크다운 본문, 갤러리, 참가자 후기 섹션
- OG 메타데이터

## 4. API 라우트

| 엔드포인트 | 메서드 | 기능 |
|------------|--------|------|
| `/api/chatbot` | POST | Gemini 스트리밍, Knowledge RAG, 메시지 저장 |
| `/api/session` | POST | 워크샵 세션 생성 |
| `/api/items` | POST | 아이템 등록 (category_id + tags upsert) ✅ v0.4 |
| `/api/comments` | GET/POST | 댓글 조회·작성 |
| `/api/categories` | GET | 카테고리 트리 반환 (5분 캐시) ✅ v0.4 |
| `/api/tags` | GET | 태그 자동완성 검색 (?q=) ✅ v0.4 |
| `/api/ontology/graph` | GET | 노드·엣지 서브그래프 (?type, ?q, ?limit) ✅ v0.4 |
| `/api/ontology/node/[id]` | GET | 노드 상세 + 1-hop 이웃 ✅ v0.4 |
